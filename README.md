# L0

[![Build Status](https://travis-ci.org/eudoxia0/l0.svg?branch=master)](https://travis-ci.org/eudoxia0/l0)

A minimal Lisp with linear types. Not yet finished.

Example:

~~~bash
$ make l0
$ ./l0 examples/fib.lisp fib.c
$ gcc fib.c
$ ./a.out
fib(30) = 832040
~~~
